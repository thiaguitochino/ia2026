# que es el machine learning? tipos de ML (diferentes aprendizajes)

1) ¿Qué es la Inteligencia Artificial (IA)? 🤖
La Inteligencia Artificial es una rama de la informática que busca crear sistemas o programas capaces de realizar tareas que normalmente requieren inteligencia humana, como aprender, razonar, reconocer imágenes, entender lenguaje o tomar decisiones.

En otras palabras, la IA intenta que las máquinas “piensen” o actúen de forma similar a las personas usando datos y algoritmos.

Ejemplos de IA:

Asistentes virtuales (como Siri o Alexa)

Reconocimiento facial

Recomendaciones de Netflix o Spotify

Autos autónomos

2) ¿Qué es el Machine Learning? 🧠💻
El Machine Learning (aprendizaje automático) es una parte de la Inteligencia Artificial que permite que las computadoras aprendan a partir de datos sin ser programadas paso a paso.

Es decir, el sistema analiza muchos datos, encuentra patrones y mejora su funcionamiento con la experiencia.

Tipos de Machine Learning (tipos de aprendizaje)

1. Aprendizaje supervisado
El algoritmo aprende usando datos que ya tienen la respuesta correcta.
Ejemplo: entrenar un sistema con fotos etiquetadas como “perro” o “gato” para que aprenda a diferenciarlos.

2. Aprendizaje no supervisado
El algoritmo recibe datos sin etiquetas y debe encontrar patrones o relaciones por sí mismo.
Ejemplo: agrupar clientes con comportamientos de compra similares.

3. Aprendizaje por refuerzo
   El sistema aprende mediante prueba y error, recibiendo recompensas o castigos según sus acciones.
Ejemplo: una IA que aprende a jugar un videojuego mejorando cada vez que gana puntos.
